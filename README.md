# GVHMR Maya

Maya importer for GVHMR bundles generated by ComfyUI-MotionCapture.

The importer reads a bundle folder, imports the SMPLX-55 skeletal rig payload
when present, can import the SMPC `.bin` as a diagnostic animated mesh cache,
and creates an animated Maya camera from `camera_maya.json`.
It can also import the experimental SMPLX-55 skeletal rig payload generated by
the `GVHMR Bundle for Maya` ComfyUI node.

## ComfyUI Setup

Use the `GVHMR Bundle for Maya` node in ComfyUI-MotionCapture.

For direct rigged Maya import, connect `GVHMR Inference / npz_path` to
`GVHMR Bundle for Maya / npz_path`. The bundle will include
`smplx_rig_maya.json`, which this importer converts into a skinned SMPL body
mesh and SMPLX-55 joint hierarchy with animation.

For a Maya-native diagnostic import that avoids retargeting, connect
`SMPL Viewer with Camera / mesh_file` to `GVHMR Bundle for Maya / smpc_bin_path`
and enable `allow_smpc_mesh`. In Maya, check `Import SMPC animated mesh cache`.
This creates a correct animated mesh preview, but it is not a rigged skeleton.

## Install In Maya

Run this once in Maya's Python Script Editor, changing the path to your local
clone of this repository:

```python
import runpy
runpy.run_path("/Users/leo/Desktop/Crafty/code/GVHMR_maya/install_maya.py")
```

The installer writes a Maya module file to your user Maya modules folder and
creates a `GVHMR` shelf button. After that, click the shelf button to open the
importer.

You can keep using the shelf immediately. Restarting Maya later verifies the
module is being picked up from a fresh session.

## Run Manually In Maya

Add this repository folder to Maya's Python path, or paste this into Maya's
Script Editor after adjusting the path:

```python
import sys
repo = "/mnt/share/dev-lbringer/markerless_mocap/GVHMR_maya"
if repo not in sys.path:
    sys.path.append(repo)

import gvhmr_maya_importer
gvhmr_maya_importer.show()
```

Then select the ComfyUI bundle folder and click `Import GVHMR Bundle`.

## Bundle Contents Used

- `manifest.json`
- `camera_maya.json`
- SMPLX skeletal rig from `files.maya_smplx_rig_json`, if present
- SMPC mesh cache from `files.smpc_bin`, if present
